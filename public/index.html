API OVERVIEW

Danh sách endpoint chính

Các module lõi hiện có trong dự án.

Authentication

  • POST /api/register — Đăng ký tài khoản
  • POST /api/login — Nhận JWT

Products

  • GET /api/products — Danh sách sản phẩm
  • POST /api/products — Thêm sản phẩm (admin)
  • PUT /api/products/:id — Cập nhật
  • DELETE /api/products/:id — Xoá

Cart

  • GET /api/cart — Lấy giỏ hàng
  • POST /api/cart/items — Thêm sản phẩm
  • PUT /api/cart/items/:itemId — Sửa số lượng
  • DELETE /api/cart — Làm trống giỏ

BEST PRACTICES

Lưu ý khi sử dụng API

  • Luôn gửi header Authorization: Bearer <token> cho các endpoint private.
  • Token có hiệu lực 100 ngày. Khi hết hạn, đăng nhập lại để lấy token mới.
  • Đối với thao tác admin (POST/PUT/DELETE sản phẩm) cần role admin.
  • Giỏ hàng tự động khởi tạo khi user thêm sản phẩm đầu tiên.

Dùng Postman?

  1. Đăng nhập và copy token.
  2. Tạo collection với biến môi trường {{base_url}}.
  3. Thiết lập header mặc định Authorization.
  4. Tận dụng file HUONG_DAN_TEST_API.md để tham khảo payload.
